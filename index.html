Playing nice with Webmaps

This project is intended to help you learn about how to interact with ArcGIS Online Webmaps. Right now it is very basic and doesn't try to cover everything. As a result - you will notice certain Webmaps don't play nicely with it.

Demo

You can see a live demo of the app here

Features

  • Search ArcGIS Online for Webmaps
  • View a list of results (currently only top 10)
  • Add a result to the map

Feedback

Please feel free to use this sample as you see fit. If you build upon it, submit a Pull Request!
